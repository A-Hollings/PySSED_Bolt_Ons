These are a collection of scripts that are designed to be run on the output files of the PySSED (Python Stellar Spectral Energy Distributions) to enhance certain data visualisation aspects of the project. The GitHub repo for this project can be found here https://github.com/iain-mcdonald/PySSED. To be clear, PySSED is not my work and I can take no credit for it. I simply got a chance to work on part of its later stages of development and incorporate some new data. The sample data file that is included in my repo here was produced by me running PySSED. 

The main aim of the bolt-ons here are to collate more of the data into the H-R diagram that is output by PySSED. For example, by making the H-R diagram interactive by means of Plotly, you can include more information about each datapoint. This makes it easier to inspect anomalous results and approach troubleshooting. Another aspect to be combined was that of the SEDs (Spectral Energy Distribution) that were produced. These are now displayed on overlay of the star in the H-R diagram where available, making it simpler to inspect the individual star for features (e.g. potential extinction due to dust clouds). These bolt-ons are still a work in development and are being added to and redesigned a little over time. These by no means reflect the intentions of PySSED's author and are of my own making.
